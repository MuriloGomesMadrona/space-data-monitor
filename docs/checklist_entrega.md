# Checklist de Entrega

Use este checklist antes de enviar sua atividade.

## Pipeline

- [ ] A pipeline executou sem erro.
- [ ] O checkout do repositório foi realizado.
- [ ] O Docker build foi executado.
- [ ] A imagem foi publicada no Azure Container Registry.
- [ ] O Azure Container Instance foi criado ou atualizado.
- [ ] A porta 8080 foi liberada.
- [ ] A URL pública foi exibida no log.

## Aplicação

- [ ] A aplicação abriu no navegador.
- [ ] O dashboard carregou os dados.
- [ ] Os gráficos foram exibidos.
- [ ] O mapa foi exibido.
- [ ] O grupo tirou print da aplicação funcionando.

## Evidências

- [ ] Print da pipeline.
- [ ] Print da aplicação.
- [ ] Link público da aplicação.
- [ ] Arquitetura desenhada.
- [ ] Breve explicação da solução.
